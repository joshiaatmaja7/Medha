# Medha
an app that supports you ADHD mind and helps you to focus and work w it and makes you feel comfortable in this world as you are.. 
My app is a smart productivity and mental support tool designed to help users manage their daily tasks while also understanding their mental state. Unlike regular to-do apps, it not only allows users to organize and track their work but also provides intelligent guidance through a built-in chat system. The app analyzes user input to detect stress, distraction, or overload and then responds accordingly, either by asking thoughtful questions in “Sherlock Mode” or giving direct step-by-step suggestions in “Assistant Mode.” It also includes features like mind mapping to organize thoughts visually and a focus mode to improve concentration. Overall, the app acts as a thinking partner that helps users reduce confusion, make better decisions, and stay productive in a simple and effective way.
